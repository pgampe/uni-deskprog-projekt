# uni-deskprog-projekt
Mensch-ärger-dich-nicht in ELM

Es soll Mensch-Ärger-Dich-Nicht in der klassischen Variante
für vier Spieler umsetzen. Dabei sollen Spieler 2-4 auch als
(einfacher) Bot angeboten werden.
